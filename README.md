# E-commerce Seller Skills for Claude 🛒

**Turn Claude into your e-commerce growth analyst.** [Claude Skills](https://docs.claude.com/en/docs/claude-code/skills) for Amazon, Shopify & Etsy sellers — optimize listings, mine reviews, and fix ad spend.

For solo sellers and small brands who do their own listings, research, and PPC.

---

## ✨ What's inside (free core)

| Skill | What it does |
|-------|--------------|
| **listing-optimizer** | Product → keyword-rich, benefit-driven title/bullets/description + backend keywords + conversion fixes, tailored to the marketplace's rules. |
| **review-analyzer** | Reviews (yours or a competitor's) → what customers love, top complaints ranked by frequency, and a product/listing action list. Find why you're getting returns. |
| **ad-campaign-analyzer** | PPC/ads export → ACoS/ROAS breakdown, **exact wasted spend**, and cut/scale verdicts per keyword. |

Each skill is a single `SKILL.md` — drop it in `~/.claude/skills/` and Claude uses it automatically.

---

## 🚀 Quick start
1. Copy a skill into `~/.claude/skills/`.
2. Ask Claude: *"Optimize this Amazon listing for conversion and search."*
3. Get a marketplace-ready rewrite.
